One Line Block

 

The One Line Block provides users with single lines of text that can be inserted into any logbook entry. These are useful for short, commonly used bits of text. This block allows multiple lines of text, enclosed in the One Line Block elements, as in this example, which provides several different individual lines of text that users can insert into their logbook entries:

 

>>>*One_Line*<<<

This entry was made in Laboratory:

This entry was made in Room:

Other personnel present were:

Experienced the following fault:

>>>*One_Line*<<<

 

Only the lines within the first instance of the One Line Block in each logbook type file will be used. All additional One Line Blocks will be ignored. This block is optional.
