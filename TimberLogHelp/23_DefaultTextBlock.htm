Default Text Block

 

The Default Text Block allows the author to provide a block of text that will be included in every logbook entry by default. When a new logbook entry is made, this text can be modified by the user, but it is always provided initially by default. This block allows multiple lines of text, enclosed in the Default Text Block elements, as in this example, which provides a header with the company name, a spot for users to add their Employee ID number within every logbook entry, and adds some additional formatting:

 

>>>*Default_Text*<<<

##################################

##################################

###   M. D. Engineering Corp   ###

##################################

##################################

Employee ID Number:

----------------------------------

>>>*Default_Text*<<<

 

Only the lines within the first instance of the Default Text Block in each logbook type file will be used. All additional Default Text Blocks will be ignored. This block is optional.
