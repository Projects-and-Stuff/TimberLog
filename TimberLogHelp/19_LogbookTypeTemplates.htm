Logbook Type Templates

 

Logbook Types are barebones structures that provide a foundation for the creation of any sort of logbook. Each Logbook Type is defined in a plain text file ending in “.logt”. The file description is very simple and flexible, and makes use of a series of Blocks, which are lines of text enclosed within Block elements. The following sections will describe these blocks, concluding with a couple of example Logbook Type files to get you started with creating your own.
