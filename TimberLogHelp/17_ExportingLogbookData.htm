Exporting Logbook Data

If the Logbook settings allow, the entire Logbook or selected Logbook Entries can be exported to the following formats:

-          Portable Document Format (PDF)

-          Text File

-          HTML (Webpage)

 

In order to export only a subset of Logbook Entries, perform a search for the relevant entries and then select the ‘This Logbook’ menu, and choose ‘Current View’ from the appropriate export format option.

 

 

Please keep in mind that very large Logbooks may take longer to format and export.
