Default Categories Block

 

The Default Categories Block allows the author to provide a list of categories that can be included in a logbook by default. When a new logbook is made, these categories can be modified for a particular logbook. Every line within this block should contain only one category name. This block allows multiple lines of text, enclosed in the Default Categories Block elements, as in this example, which provides six different default categories:

 

>>>*Default_Categories*<<<

Electrical Engineering

Computer Engineering

Personal Projects

Mechatronics

Drawing Notes

Regulations

>>>*Default_Categories*<<<

 

Only the lines within the first instance of the Default Categories Block in each logbook type file will be used. All additional Default Categories Blocks will be ignored. This block is optional.
