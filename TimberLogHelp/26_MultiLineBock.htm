Multi Line Bock

 

The Multi Line Block provides users with multiple-line chunks of text that can be inserted into any logbook entry. These are useful for longer, commonly used chunks of text. It allows the logbook type author to standardize entry formatting. This block allows multiple lines of text, enclosed in the Multi Line Block elements, as in this example, which provides three different Multi Line chunks of text that users can insert into their logbook entries:

 

>>>*Multi_Line*<<<

Major Fault

Type of Fault:

Location of Fault:

Symptoms:

Cause:

Has the Fault been corrected:

If yes, how:

>>>*Multi_Line*<<<

 

 

 

>>>*Multi_Line*<<<

Experiment

Experiment Name:

Location conducted:

 

Description of Experiment:

 

Duration of experiment:

Equipment used:

 

End Result:

 

Conclusions:

 

References:

>>>*Multi_Line*<<<

 

 

 

>>>*Multi_Line*<<<

Project Update

Project Subcomponent:

 

Work Performed:

 

 

Results and Conclusions:

>>>*Multi_Line*<<<

 

Multiple instances of the Multi Line Block are allowed in each logbook type file. The first line will be used as the title to help the user identify its purpose, and is not inserted into the logbook entry. These blocks are optional.
