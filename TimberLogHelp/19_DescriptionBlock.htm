Description Block

 

The Description Block allows the author to provide details about the logbook type: what it is to be used for, what features it includes, who should use this logbook type, etc. This block allows multiple lines of text, enclosed in the Description Block elements, as in this example, which describes some details about a logbook type:

 

>>>*Description*<<<

This logbook type can be used for a variety of purposes, including Industrial Engineering, Electrical Engineering, Chemical Engineering, Mechatronics, Computer Engineering, or personal engineering projects and designs.

 

This logbook type provides a number of templates for general purpose engineering entries.

 

This logbook type was designed by James F. Park.

>>>*Description*<<<

 

Only the lines within the first instance of the Description Block in each logbook type file will be used. All additional Description Blocks will be ignored. This block is required.
