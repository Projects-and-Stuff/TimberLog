Using Logbook Automation Features

Depending on the features of the Logbook Type your Logbook was generated from, there may be a number of automation features available for use in your Logbook.

 

Default Text – If enabled, default text will automatically be shown every time a new entry is started

 

Logbook Entry Categories – If enabled, users can select a category for their logbook entries which can be used for searching

 

One-Line Templates – To insert one-line templates, right-click within the entry textbox or type CTRL+SPACE while the cursor is in the textbox. A menu will pop up with the available one-line template options. Selecting one of these and pressing enter, or double-clicking on it, will insert the line of text into the Logbook.

 

Multi-Line Templates – These templates are listed by their title in a listbox to the left of the entry textbox. Select the appropriate title, press the ‘Insert’ button below the multi-line templates listbox, and the associated text will be inserted into the Logbook entry.
