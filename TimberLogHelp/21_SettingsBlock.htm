Settings Block

 

The Settings Block allows the author to define certain settings to be used in a logbook type. Settings muse be formatted correctly and enclosed in the Settings Block elements, as in this example, which defines “Sort_Categories” as true:

 

>>>*Settings*<<<

Sort_Categories=True

>>>*Settings*<<<

 

Currently, the following settings can be defined. If a setting is not included in the Type File, the default value will be used:

Sort_Categories [default=True]
- Sorts the categories alphabetically when displayed to the user.


Sort_One_Line_Text [default=True]
- Sorts the lines within the One_Line text alphabetically when displayed to the user. If you want these lines to be presented in a particular order, set this to false.


Sort_Multi_Line_Text [default=True]
- Sorts the Multi_Line names alphabetically when displayed to the user. If you want these lines to be presented in a particular order, set this to false.


Embed_TypeFile [default=True]
- Embeds the type file into the Logbook. The benefit of this is that if the logbook is used on a different computer, a copy of the Type File can be exported to the local computer at any time, allowing creation of additional logbooks based upon the same Logbook Type.

 

Only correctly formatted lines within the first instance of the Settings Block in each logbook type file will be used. All additional Settings Blocks will be ignored. This block is optional. If not used, all settings will use their default values as described above.
