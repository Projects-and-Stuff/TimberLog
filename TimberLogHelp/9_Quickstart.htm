Quickstart

 

To quickly create a new logbook, open TimberLog and select an appropriate Logbook Type from the list on the right side of the Start Screen.

 

Fill in:

-          Logbook Name

-          A short Logbook Description

-          Your name as the person opening the logbook

-          Optionally, set a master password by checking the “Master Password Required to View or Add New Logbook Entries” checkbox and typing the new password into both text boxes.

 

Click on the “Create” option.

 

Select a file name and location to save your new logbook.

 

When the logbook window opens, write the text of the first logbook entry in the text area at the bottom of the screen, type your initials in the “Name” text box, and click “Add New Entry”. The text you typed should now appear in the upper text area.

 

Voila – you have now created your first logbook and made your first logbook entry!
