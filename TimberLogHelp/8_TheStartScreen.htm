The Start Screen

The Start Screen is normally the first thing visible when opening the TimberLog (if you’ve opened a logbook file directly from Windows Explorer, you won’t initially see the Start Screen). It is also visible any time you close a logbook.

 

The purpose of the Start Screen is to serve as starting point for opening logbooks, or for creating a new logbook.

 

When the Start Screen initially opens, the left panel allows you to choose from a list of recently opened logbooks. You can also select the “Open Other Logbook” option to navigate your computer system in order to open an existing logbook. On the right half of the Start Screen is a list of available logbook Template Types. These serve as the foundation for any logbook. There should always be at least one Logbook Type available, and you can very easily make your own Type Templates.

 

If you want to make a new logbook, once you click on one of the available Types, you will be able to select the parameters and options for the new logbook.
