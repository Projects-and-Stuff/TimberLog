Logbook Type Block

 

The logbook type Block defines the name for the logbook type. This block allows a single line of text, enclosed in the Logbook Type Block elements, as in this example, which defines the logbook type name of “Engineering Journal”:

 

>>>*Logbook_Type*<<<

Engineering Journal

>>>*Logbook_Type*<<<

 

Only the first line of the first instance of the Logbook Type block in each logbook type file will be used. All additional Logbook Type Blocks will be ignored. This block is required.
